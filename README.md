# E-Commerce React App

A simple e-commerce frontend built with React and Vite, featuring multiple product categories and navigation.

## Features

- Product listing by categories: Beauty Care, Accessories, Dress, Cosmetic
- Add to cart functionality
- Cart management
- Navigation between pages using React Router
- Responsive design

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Run the development server:
   ```
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Pages

- **Home**: All products
- **Beauty Care**: Shampoos, lotions, face masks
- **Accessories**: Jewelry, handbags, sunglasses
- **Dress**: Summer dresses, evening gowns
- **Cosmetic**: Lipsticks, foundations

## Build for Production

```
npm run build
```

## Troubleshooting

- Ensure Node.js and npm are installed.
- If port 5173 is in use, Vite will suggest an alternative port.
